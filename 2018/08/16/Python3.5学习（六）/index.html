2. 面向对象

一种编程范式，所有东西都可以按照属性分类。

2.1 基础知识

class类：

指具有相同属性对象的抽象集合，具有属性和方法。

object对象：

一个对象是一个类实例化后形成的，类必须进行实例化后才能成为具体的对象个体

2.2 三大特性

封装：

在类中对数据的赋值、内部调用等操作，对外部用户是封闭的
这样使类变成了一个胶囊或容器，里面包含类的数据和调用方法。

继承：

一个类可以派生出多个子类，在这个父类里定义的属性、方法自动被子类继承，python同时也可以实现多继承。

多态：

一个接口多个实现。
实现的接口的复用。

细说创建实例的过程
当我们将一个类实例化的时候通常写成 object = class()的形式，需要传递参数a的时候写成object = class(a)的方式传递。但我们在创建类的时候通常会在定义属性和方法的时候默认将self传入，那么self又是什么呢？
其实在实例化的时候会自动将创建的实例的内存地址传送到类中，我们在进行类的初始化的时候self.attr = attr的过程就是将需要的参数传送到实例中。因此可以说self就是实例对象

3. 实例变量与类变量

所谓实例变量就是静态属性，指的是我们在def init(self)中初始化的属性，作用域为实例本身
类变量，不通过构造函数，直接在类中定义的变量。作用域为类，特点是不通过实例化就可以调用。作用：类共有的不希望被实例修改的属性，可以节省开销。

例：

class Perinfo(object):
    "身份信息系统"
    #类变量
    country = 'China'
    def __init__(self,name,sex,*args):
        #实例变量
        self.name = name
        self.sex = sex
        if args:
            self.country = args[0]
#无需经实例化直接调用类变量
print(Perinfo.country)
p1 = Perinfo('zt','male')
p2 = Perinfo('cao','female','Amarica',)
p3 = Perinfo('wang','male')
print("%s's country is %s\n%s's country is %s\n%s's country is %s\n"\
      %(p1.name,p1.country,p2.name,p2.country,p3.name,p3.country))
'''
China
zt's country is China
cao's country is Amarica
wang's country is China
'''

1. 实例变量和类变量同名时候，初始化后会使用实例变量
2. 当通过实例来修改类变量的时候本质是创建了一个名字相同的实例变量，显示的是实例变量的值,通过其他对象调用时，类变量并没有修改
3. 只有通过类来调用类变量的时候会更改

4. 析构函数

析构函数：在实例释放、销毁的时候执行的，通常用于一些收尾工作。
如关闭一些数据库链接的缓存文件。

析构函数的写法如下：

def __del__(self):
    print("写入析构函数内容")

需要注意是在实例释放和销毁的时候执行，默认实例释放的时候是在程序执行完毕才会释放，如有需求则可以手动释放

下面来看一个项目实例：

import time
class Project(object):
    def __init__(self,name,starttime,costtime):
        self.name = name
        self.starttime = starttime
        self.costtime = costtime
    def run(self):
        starttime_str = time.strftime("%Y-%m-%d %H:%M:%S",time.localtime(self.starttime))
        print("Project %s's starttime: %s"%(self.name,starttime_str))
        finishtime_str = time.strftime("%Y-%m-%d %H:%M:%S",time.localtime(self.starttime+self.costtime))
        print("Project %s's finishtime: %s"%(self.name,finishtime_str))
    def __del__(self):
        print('Project %s has finished!!'%self.name)
p1 = Project('geography',time.time(),300)
p1.run()
#del p1
p2 = Project('game',time.time(),600)
p2.run()

在例子中我们可以看到存在一处析构函数，我们运行程序得到的结果如下：

'''
结果为：
Project geography's starttime: 2018-08-20 22:11:04
Project geography's finishtime: 2018-08-20 22:16:04
Project game's starttime: 2018-08-20 22:11:04
Project game's finishtime: 2018-08-20 22:21:04
Project geography has finished!!
Project game has finished!!
'''

可以发现程序在全部执行结束的时候才会释放实例p1与p2，但是当我们需要在每个实例完成后立刻执行析构函数的话则需要手动释放实例，如下：

p1 = Project('geography',time.time(),300)
p1.run()
del p1
p2 = Project('game',time.time(),600)
p2.run()

通过del来释放实例，得到的结果如下：

'''
结果为：
Project geography's starttime: 2018-08-20 22:20:47
Project geography's finishtime: 2018-08-20 22:25:47
Project geography has finished!!
Project game's starttime: 2018-08-20 22:20:47
Project game's finishtime: 2018-08-20 22:30:47
Project game has finished!!
'''

5. 类的继承

一个类可以派生出子类，在这个父类里定义的属性、方法自动被子类继承。
子类可以完全使用父类中的全部方法，同时子类也可以重写父类中的方法。

实例：

class People(object):
    def __init__(self,name,age):
        self.name = name
        self.age = age
        self.friends =[]
    def sleep(self):
        print("%s is sleeping..."%self.name)
    def talk(self):
        print("%s is talking..."%self.name)
    def eat(self):
        print("%s is eating..."%self.name)
#将父类放在()中
class Man(People):
    #先建了自己的name，然后在传递父类的
    def __init__(self,name,age,money):
        super(Man,self).__init__(name,age)
        self.money = money
    def talk(self):
        #调用了父类中的talk方法
        super().talk()
        print("man talk %d"%self.money)
    #重写了父类的方法
    def sleep(self):
        print('man is sleeping')
    def drink(self):
        print("%s is drinking..."%self.name)

class Woman(People):
    def __init__(self,name,age,money):
        super(Woman,self).__init__(name,age)
        self.money = money
    def spand(self,cost):
        print("%s spend money"%self.cost)
m1=Man('zt','23',4000)
w1 = Woman("cao",'18',4000)
m1.talk()
m1.sleep()
m1.drink()
w1.spand(200)

python也支持多继承，在java中是不支持的。

先来看下面一个例子：

class Person(object):
    def __init__(self,name,sex):
        self.name = name
        self.sex = sex

    def eat(self):
        print("%s is eating"%self.name)
class Job(object):
    def __init__(self,job):
        self.job = job

class Explorer(Person,Job):
    def __init__(self,name,sex,job):
        super(Explorer,self).__init__(name,sex)
        self.job = Job(job)
e1 = Explorer("zhao","男",'it')
print(e1.job.job)

该例子实现了多继承，explorer继承了person和job两个不同的类，从而实现了explorer的属性，与其说是多继承，不如说是组合，将两个独立的类中有用的属性组合成新的类的方式。

  4. 新式类中使用super(class,self).init()来进行参数的传递

  5. 使用新式类的好处是防止父类的名称进行更改时子类中也必须进行大量的更改。

这里涉及到super传递参数时候的继承顺序问题：

要理解super的原理，就要先了解super的内核mro。mro是method resolution order的缩写，表示了类继承体系中的成员解析顺序。
在python中，每个类都有一个mro的类方法。

我们通过print(Man.mro())来查看得到如下结果： 

[<class '__main__.Man'>, <class '__main__.People'>, <class '__main__.Relation'>, <class 'object'>]

可以看到mro方法返回的是一个父类的列表。Man中的每个父类都在其中出现一次，这也是super在父类中查找成员的顺序。

因此使用super来进行多继承的时候成员的访问顺序就会按照mro中的顺序来查找

在遇到初始化问题时候也可以很好的解决，当mro中的多个类存在初始化函数时，super只执行第一个初始化函数，其他的不执行。

下面看一个例子：

class A:
    def __init__(self):
        print("A")
class B(A):
    print("B中有构造函数")
    def __init__(self):
        print("B")
class C(A):
    def __init__(self):
        print("C")
class D(B,C):
    print("D中有构造函数")
    def __init__(self):
        print("D")
d1 = D()
print(D.mro())

结果为：

B中有构造函数
D中有构造函数
D
[<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>]

从结果中我们可以看到当继承父类的时候，类变量的执行顺序为先访问父类的类变量再访问子类的类变量。super的继承初始函数的顺序为DBCA，本例中D中存在构造函数，因此父类中的构造函数不被执行。

我们将本例改为：

class A:
    def __init__(self):
        print("A")
class B(A):
    print("B中无构造函数")
class C(A):
    print("C中有构造函数")
    def __init__(self):
        print("C")
class D(B,C):
    print("D中无构造函数")
d1 = D()
print(D.mro())

结果为：

B中无构造函数
C中有构造函数
D中无构造函数
C
[<class '__main__.D'>, <class '__main__.B'>, <class '__main__.C'>, <class '__main__.A'>, <class 'object'>]

本例中，CA中存在构造函数，程序执行先按照父类的类变量先访问的原则访问了类变量，之后只访问了C中的构造函数，很好的说明了super继承的访问顺序。

python3中super继承顺序是按照广度优先来继承的。
python2中super继承的顺序则是按照深度优先的原则来继承的。

python3中多用新式类，但是在新式类中使用super只能传递一个父类变量，如果想要传递多个父类的变量则有两种方法

1，使用经典类，引用多继承的第一个例子：

7. 继承实例

建立学校系统，储存老师和学生信息，可以处理学生入学和老师入职的操作。老师和同学都需要保存基本的个人信息，不同的是老师需要有薪资和教授的课程，学生需要学号和年级信息。可以查看老师教授的课程，提醒学生交学费的方法。

#创建学校系统类
class School(object):
    def __init__(self,name,addr):
        self.name = name
        self.addr = addr
        #存储学生对象的列表
        self.students = []
        #存储员工对象的列表
        self.staffs = []
    #学员入学
    def enroll(self,stu_obj):
        print("为学员%s办理注册手续"%stu_obj.name)
        self.students.append(stu_obj)
    #员工入职
    def hire(self,staffs_obj):
        print("雇佣%s为新员工"%staffs_obj.name)
        self.staffs.append(staffs_obj)
#创建学校成员类
class SchoolMember(object):
    def __init__(self,name,age,sex):
        self.name = name
        self.age  = age
        self.sex = sex
    def tell(self):
        pass
#创建教师类，继承自学校成员类
class Teacher(SchoolMember):
    def __init__(self,name,age,sex,salary,course):
        #将从父类中查找到的变量赋给Teacher的self中
        super(Teacher,self).__init__(name,age,sex)
        self.salary = salary
        self.course = course
    #重写了tell方法
    def tell(self):
        print("""
        ---------- info of Teacher:%s ----------
        Name:%s
        Age:%s
        Sex:%s
        Salary:%s
        Course:%s
        """%(self.name,self.name,self.age,self.sex,self.salary,self.course))
    #添加了teach方法
    def teach(self):
        print("%s is teaching %s"%(self.name,self.course))
class Students(SchoolMember):
    def __init__(self,name,age,sex,stu_id,grade):
        super(Students,self).__init__(name,age,sex)
        self.stu_id = stu_id
        self.grade = grade

    def tell(self):
        print("""
        ---------- info of Student:%s ----------
        Name:%s
        Age:%s
        Sex:%s
        Stu_id:%s
        Grade:%s
        """%(self.name,self.name,self.age,self.sex,self.stu_id,self.grade))

    def pay_tuition(self,amount):
        print("%s has paid tuition for %s"%(self.name,amount))
#创建对象
sc1 = School("东北大学","沈阳")
t1 = Teacher("Wen",34,"M",2000000,"Linux")
t2 = Teacher("Cui",23,"M",300000,"Python")

8. 多态&静态方法

所谓多态就是一种接口多种实现,python并没有明确解释多态的变量，但有实现的方法。
下面来看一个例子：

class Animal:
    def __init__(self,name):
        self.name = name
    def talk(self):
        pass
class Cat(Animal):
    def talk(self):
        print("Meow")
class Dog(Animal):
    def talk(self):
        print("Woof")
def animal(obj):
    obj.talk()
d = Dog("Dog")
c= Cat("Cat")
animal(d)
animal(c)

我们可以看出只要用一个函数就可以实现多态的方法。
但是我们需要将一些方法进行整合，比如动物不仅仅有叫这个方法，动物还有吃，睡等方法，我们需要将这些方法加入到一个类中。
方法如下：

class Animal:
    def __init__(self,name):
        self.name = name
    def tell(self):
        pass
    @staticmethod
    def animal_talk(obj):
        obj.talk()
class Cat(Animal):
    def talk(self):
        print("Meow")
class Dog(Animal):
    def talk(self):
        print("Woof")
d = Dog("Dog")
c= Cat("Cat")
Animal.animal_talk(d)
Animal.animal_talk(c)

可以看到用上面的方法就可以将方法写在类中，然后通过类来调用这个方法达到接口的目的。
在上面例子中用到的@staticmethod就是静态方法。

静态方法有如下特性：

  1. 实际上和类没什么关联，只相当于函数
  2. 用途相当于一个在类中组合在一起的一些功能
    　例如os模块,os.mkdir与os.system()并没有什么关联但是统一封装在os模块中
