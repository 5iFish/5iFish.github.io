函数式编程

函数式编程思维更接近数学计算

函数式编程是一种抽象程度很高的编程范式，纯粹的函数式编程
是没有变量的，因此，任意一个函数，只要输入是确定的输出
就是确定的，这种纯函数我们称之为没有副作用。

举例：

（1+2）*3-4

传统编程：

var a = 1+2;
var b = a*3;
car c = b-4;

函数式编程：

var result = subtract(multiply(add(1,2),3),4)

函数式编程不是函数

函数

1.面向对象：类 class
2.面向过程：过程 def
3.函数式编程：函数 def

python中函数定义方法：

def test(x):
    "The function definitions"
    x+=1
    return x
'''
def:定义函数的关键字
test;函数名
():定义形参
"":文档描述（非必要，建议为函数添加描述信息）
x+=1:泛指代码块或程序处理逻辑
return:定义返回值
'''

递归

递归三个条件：

  1. 递归要有明确的条件
  2. 每次进入更深层递归时，问题规模相比上一次应该有所减少
  3. 递归效率不高，递归层次多会导致栈溢出

例子：

没有明确的终止条件：

def calc(n):
    print(n)
    return calc(n/2)
calc(10)

正确的递归表达：

def calc(n):
    print(n)
    if int(n/2)>0:
        return calc(int(n/2))
    print('->',n)
calc(10)
'''
结果为：
10 5 2 1 ->1
'''

高阶函数

变量可以指向函数。函数的参数能接收变量，那么一个函数就能接收另一个函数作为参数，这种函数成为高阶函数

def add(x,y,f):
    return f(x)+f(y)
res = add(3,-6,abs)
print(res)
